# TeamControl

Instruções de Instalação
1. Execute: composer install / php composer.phar install
2. Execute no seu banco de dados (MariaDB/MySQL), crie um novo DB com o nome teamcontrol, e faça a restauração utilizando
o arquivo db_teamcontrol.sql
